# 변수와 상수
## 변수
### 개념
- **변수**: 합계나 계산에 사용되는 숫자 또는 문장의 일부로 사용되는 문자열과 같은 값을 담는 컨테이너
	=> 변수는 값을 포함. 값 자체가 아닌 **값을 담는 컨테이너**
	
- 선언: 변수를 만드는 행위
```javascript
let MyName;
```
- 초기화: 변수를 선언한 후에는 값으로 초기화할 수 있음.
```javascript
MyName = "Eunkyoung";

//선언과 동시에 초기화도 가능
let myCat = "Maru";
```
### 명명 규칙
- 변수명에는 오직 문자와 숫자, `$`와 `_`만 들어갈 수 있음
- 첫 글자가 숫자가 될 수는 없음
#### 변수명 표기 법
- 카멜 케이스(camelCase): 띄어쓰기를 하지 않고 각 단어의 첫 글자를 대문자로 붙여 쓰되, 전체 단어의 첫 글자는 대문자 또는 소문자로 쓸 수 있는 방식 => javascript에서는 카멜 케이스를 주로 사용
- 스네이크 케이스(snake_case): 띄어쓰기 대신 밑줄을 사용하며 모든 단어를 대체로 소문자로 작성하는 방식
## 상수
### 개념
- 상수: 변화하지 않는 변수, 프로그래머가 변경할 수 없는 값
	=> 상수를 선언할 땐, `let`대신  `const`를 사용
```javascript
const myCountry = "Korea";
```


# 자료형
- 자바스크립트는 변수에 저장되는 값의 타입을 언제든지 바꿀 수 있는 언어로 동적 타입(dynamically typed) 언어이다
### 숫자형
- 숫자(정수)나 2.456(부동소수점 또는 부동 소수점 숫자) 등의 숫자를 나타낼 때 사용
```javascript
let myNumber = 17;
```
### 문자형
- 빈 문자열이나 글자들로 이루어진 문자열을 나타낼 때 사용
- 문자열 값을 지정할 때는 따옴표로 묶어야 함
	- 큰 따옴표: `"Hello"`
	- 작은 따옴표: `'Hello'`
	- 역 따옴표(백틱, backtick): ![[Pasted image 20250910100250.png]]
	=> 큰 따옴표와 작은 따옴표는 기본적인 따옴표로 자바스크립트에서는 둘의 차이를 두지 않고 있고, 역따옴표로 변수가 표현식을 감싼 후 `${...}`안에 넣어주면, 원하는 변수나 표현식을 문자열 중간에 넣을 수 있음
```javascript
let name = "Kim"

// 변수를 문자열 중간에 삽입
alert(`Hello, ${name}!`); // Hello, Kim
```
### 불린형
- `true`와 `false` 두 가지 값 밖에 없는 자료형
	- 긍정이나 부정을 나타내는 값을 저장할 때 사용
	- 비교 결과를 저장할 때도 사용
```javascript
let nameFieldChecked = true;
let test = 4 > 1;
```
### null과 undefined
- `null`: 자바스크립트에서는 존재하지 않는 값, 비어있는 값, 알 수 없는 값을 나타낼 때 사용
- `undefined`: 값이 할당되지 않은 상태
```javascript
// null
let age = null;

// undefined
let age;
alert(age); // undefined 출력
```
